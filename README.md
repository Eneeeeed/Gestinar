Gestinar offline app.
Gestinar es una aplicación offline que busca facilitar el control y gestión de inventarios de Materia
prima o insumos, con el objetivo de evitar realizar pedidos de estos durante etapas ya tardias donde
no se cuente con estos elementos para continuar con el proceso productivo.

Alcance del sistema

-Aplicación local (offline)

-Uso interno por un solo equipo o área

-Enfoque en control de inventario y trazabilidad de movimientos

-Exportación de información para análisis externo

¿Que información requiere el sistema?
Este al ser un sistema basico, no requerira datos relacionados con el tipo de empresa, RUC, etc. y tambien
al ser un sistema solo enfocado en un equipo, no requerira de logins u otros parecidos, porque, esta enfocado
en una sola area de la empresa. Cabe mencionar que esto puede llegar a cambiar con el tiempo.

Para que se pueda realizar la gestión y control, la información necesaria por ingresar sera el nombre del item,
descripción del item (si es necesario), cantidad de existencias y unidad de media.

Para el control de ingresos, sera necesario que se ingrese el item ingresado, cantidad por entrar, proveedor,
fecha de pedido y fecha de llegada de pedido

Para el control de salidas, sera necesario que se ingrese el item consumido, cantidad por consumir, operario que
realizo la solicitud y fecha de consumo.

¿Que se espera que de el sistema?
Al recibir la información mencionada anteriormente, se espera que para el control de operarios, se muestre un
archivo excel general con el stock actual de los items, sus nombres, codigo (autogenerados) y unidad de media.
Tambien se espera que se presente dentro de este mismo excel cuales fueron las entradas y salidas de cada uno de
los items.

Esta información se mostrara tanto dentro (la propia app) y fuera del sistema (excel).
